# 🎓 Student Management System (Python Project)

A complete **Student Management System** built using Python.  
This project is designed as a beginner-to-intermediate level program that demonstrates how real-world applications manage and process data.

It is a **menu-driven console application** that allows users to perform full CRUD operations (Create, Read, Update, Delete) on student records.

---

## 🚀 Project Overview

This system helps manage student information such as:

- Name
- Roll Number
- Marks (out of 300)
- Percentage

The project simulates a basic real-world school management system using Python.

---

## ✨ Features

### 📌 Core Features
- ➕ Add new student records
- 📄 View all students
- 🔍 Search student by roll number
- ✏️ Update student details
- ❌ Delete student records
- 🚪 Exit program safely

### 🧠 Smart Features
- Prevents duplicate logic errors
- Clean menu-driven interface
- Real-time calculation of percentage
- Loop-based continuous system until exit
